DICOM Medical Image Viewer

Windows

Double-click viewer-win32.exe to open the DICOM viewer.
If autorun is enabled, the viewer will start automatically when the disc is inserted.

Linux

Open a terminal in the disc directory and run: ./viewer-linux.sh

macOS

Open viewer-mac.app
Note: This disc contains a portable DICOM viewer (Weasis) with an embedded Java Runtime. No installation is required. If the viewer is slow to load from disc, you may copy the entire disc contents to a local folder and run from there for better performance.

Modifications

The following modifications were made to the original Weasis Portable distribution to enable reliable playback from optical media (DVD/CD):

  • Custom launcher script (start-weasis.bat) — replaces the original Launch4j wrapper to support the disc folder layout, auto-detect 32/64-bit architecture, and select the appropriate Java Runtime.
  • Dual JRE bundling — both 32-bit and 64-bit Java Runtimes are included for maximum compatibility across Windows systems.
  • Fast-launch mechanism — copies viewer files to a local temporary folder before launch, avoiding slow random reads from optical media.
  • DICOMDIR excluded — the viewer scans the DICOM folder directly, which is more reliable than the path-dependent DICOMDIR index.

These are launcher and packaging changes only. The Weasis viewer application itself, all Java libraries, and the Java Runtime are unmodified original distributions.
